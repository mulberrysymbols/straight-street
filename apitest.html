Somehting.com api test

This example web application demonstrates how the somehting.com api can be used in a client side web app. It provides example code that exercises the api and demonstrates the use some of the API features. This is in no way intended to be a complete application, or be without issues that would be resolved in a released solution.

Hint: try searching for 'sweet' or 'food'.

For each symbol a thumbnail GIF is provided and scalable SVG or WMF formats are available if your browser supports them; IE does not support SVG, Safari and Firefox do not support WMF and Safari seems to incorrectly draw SVGs fullsize. The scalable SVG and WMF provide links to fullsize images.
